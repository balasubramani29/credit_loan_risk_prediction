# 💳 Credit Risk Prediction Dashboard

A machine learning-powered web application that predicts whether a customer is at **High Risk** or **Low Risk** for credit approval based on financial and personal information.

## 🚀 Live Demo

🔗 **App Link:** https://creditloanriskprediction-66rxicezvuvkhphpkzjdus.streamlit.app/

---

## 📌 Project Overview

The Credit Risk Prediction Dashboard uses a **Random Forest Classifier** to analyze customer information and predict credit risk. The application provides an interactive dashboard with model performance metrics, visualizations, and real-time predictions.

---

## ✨ Features

* 📂 Dataset preview and summary statistics
* 🤖 Credit risk prediction using Machine Learning
* 📈 Model accuracy and performance metrics
* 🔍 Confusion Matrix visualization
* ⭐ Top 10 Feature Importance chart
* ✍ Interactive sidebar for customer data input
* 💻 Professional and user-friendly Streamlit interface

---

## 🛠 Technologies Used

* Python
* Streamlit
* Pandas
* NumPy
* Matplotlib
* Scikit-learn

---

## 📊 Machine Learning Workflow

1. Load and preprocess the credit risk dataset
2. Handle categorical variables using encoding techniques
3. Split data into training and testing sets
4. Train a Random Forest Classifier
5. Evaluate model performance
6. Predict customer credit risk through an interactive interface
